Pololu Micro Metal Gearmotor 3D (STEP) Models

File name key:

pm = precious metal brushes (LP, MP, and HP versions)
cb = carbon brushes (HPCB versions)

1000 = 1000:1 gear ratio, which has a longer gearboxes than the other gear ratios

enc-back-con = Encoder with Back Connector version
enc-side-con = Encoder with Side Connector version
ext-shaft = Extended Motor Shaft version

with-1086-bracket = gearmotor shown with Pololu item #1086 bracket (https://www.pololu.com/product/1086) and also covers #989 bracket which is the same but black (https://www.pololu.com/product/989)
with-1089-ext-bracket = gearmotor shown with Pololu item #1089 bracket (https://www.pololu.com/product/1089)
